# 🔧 LINKO Backend – Real-Time Chat & Video API

Backend-ul aplicației LINKO este construit cu **FastAPI** și oferă suport complet pentru autentificare securizată, mesagerie în timp real și apeluri video prin WebRTC.

---

## 🚀 Tech Stack

- ⚡ FastAPI (Python)
- 🗄️ SQLite (SQLAlchemy ORM)
- 🔐 JWT Authentication
- 🔑 Bcrypt (hash parole)
- 🔌 WebSockets (real-time communication)
- 🎥 WebRTC Signaling
- 📦 Pydantic (data validation)
